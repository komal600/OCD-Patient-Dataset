# OCD-Patient-Dataset using AI
Project Overview
This project explores patient data related to Obsessive-Compulsive Disorder (OCD) using Excel. The goal is to organize, analyze, and visualize health records to identify patterns in demographics, symptoms, and treatment outcomes.
Key Features
- Pivot Table Analysis: Summarized patient data across age groups, gender, and treatment categories.
- Visual Dashboards: Designed charts to highlight patient distribution, progress trends, and outcome comparisons.
- Excel-Only Workflow: All analysis and visualization performed within Excel for simplicity and accessibility.
Tools & Techniques
- Microsoft Excel (Pivot Tables, Calculated Columns, Charts)
